Nise (偽)
---

Nise is a translator from Japanese to Chinese-like fake (hanamogera) string.

### Usages

```shellsession
$ nise
学校に行きます
学校行了
```

```shellsession
$ echo 学校に行きます|nise
学校行了
```

```shellsession
$ nise 学校に行きます
学校行了
```

```shellsession
$ nise --help
nise is a translator from Japanese to Chinese-like fake (hanamogera) string. see github.com/ikawaha/nise
exit status 1
```
